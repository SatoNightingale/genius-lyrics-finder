Este programa recorre todos los archivos de audio mp3 en una carpeta y busca sus letras en el sitio web [genius.com](https://genius.com/), guardándolas en el archivo si este no tiene letras asignadas. Cuenta con una interfaz gráfica donde se puede ver el proceso a medida que ocurre así como información de cada canción

Mejoras añadidas en esta versión:

- Ahora hay un botón para volver a buscar las letras de las canciones seleccionadas en la lista
- Se pueden modificar los datos de título y artista de la canción a buscar
